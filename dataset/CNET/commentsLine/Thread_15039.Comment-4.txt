format the drive using the full method.
then open a command prompt and run &quot;cipher /w x:\ &quot; (without the quotes, where x: is the drive letter of the drive in question.
this works for windows os'es from windows 2000 onward.
for vista, windows 7 (and i assume, windows 8) you will probably have to run cmd as an administrator.
this method uses tools (format and cipher) available within the os; no need to download anything!
