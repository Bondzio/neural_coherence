<s> I get exactly the same error with this card (cannot start code 10) on an XP SP2 desktop. </s>
<s> I downloaded and installed the latest drivers from the Belkin website with no result, so if anyone does know how to persuade them to work... </s>
<s> Go to Start/Administrative Tools/Services. </s>
<s> Find the Wireless Zero Config Service and DISABLE it. </s>
<s> Reinstall your F5D7011 software and it should work fine. </s>
<s> The problem is that Windows tries to configure the card and screws it up. </s>
