go into the bios to your video settings.
when you enter, it should have the onboard option, agp options, and an auto configuration.
when you enter into the agp options, it should have speeds indicated.
if it simply says agp, or if you can select auto, a 4x/8x will do fine.
if you install an agp card, disable the onboard graphics controller in the windows device manager prior to installing the card.
