<s> WPA has a number of improvements over WEP, the most significant being the use of Temporal Key Integrity Protocol (TKIP), which dynamically changes keys as the system is used. </s>
<s> This provides greatly improved protection against, and effectively defeats, the well-known key recovery attacks on WEP. </s>
<s> Your question suggests you will be using Personal WPA, which utilizes pre-shared key (PSK) mode. </s>
<s> This is where every allowed computer is given the same passphrase. </s>
<s> In PSK mode, security depends on the strength and secrecy of the passphrase. </s>
<s> The passphrase may be from 8 to 63 printable ASCII characters or 64 hexadecimal digits (256 bits). </s>
<s> However, the weak passphrases that users typically employ are vulnerable to password cracking attacks. </s>
<s> To protect against a brute force attack, a truly random passphrase of at least 20 characters should be used, and 33 characters or more is recommended. </s>
<s> In summary the more random your WPA preshared key, the safer it is to use. </s>
