<s> Many of the Wi-Fi router companys make a print server which is Wi-Fi enabled. </s>
<s> You plug the printer to the print server and in turn the printer server will communicate to the router wirelessly. </s>
<s> Each computer that will use the print server will need the device driver installed and you or somebody of your choosing can install the device drivers (the instructions that each component needs in order for each to talk to each other). </s>
<s> Those device drivers are supplied in the box with the print server that you purchased. </s>
<s> As the previous poster said, follow the manufacturers instructions and don't skip or &quot;think&quot; you know more than they. </s>
